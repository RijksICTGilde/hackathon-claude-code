# Oefeningen

Leren gaat het best door experimenteren in een afgeschermde sandbox. Deze oefeningen laten je de mogelijkheden van Claude Code op je eigen tempo ontdekken — je pakt wat bij jouw niveau past. Alle oefeningen draaien om concrete experimenten op coding-challenges, niet om theorie. Voor inspiratie en bronnen: [codingchallenges.fyi](https://codingchallenges.fyi/challenges/intro) biedt een scala aan challenges, en de [cc.storyfox.cz cheat sheet](https://cc.storyfox.cz/) is een handig naslagwerk voor Claude Code-features.

## Vier categorieën

Kies waar je mee aan de slag wil — je ervaring met Claude Code bepaalt mede je instap.

1. **[Werken met Claude Code](oefeningen/claude-code.md)** — slash-commands, plan mode, `--dangerously-skip-permissions`, subagents, zelf-review, etc.
2. **[Plugins, skills en tools](oefeningen/plugins-skills-tools.md)** — wat de meegeleverde overheid-marketplace, `superpowers`, `rtk`, `ralph-loop` en andere features toevoegen aan een kale Claude Code-installatie.
3. **[Prompting-technieken](oefeningen/prompting.md)** — informeel vs spec-driven vs structured prompt-driven, beproefd op dezelfde challenge.
4. **[Voor wie verder wil](oefeningen/verbeteringen.md)** — open einde voor gevorderde deelnemers; experimenteer met iets nieuws en draag het terug.
